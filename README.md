## 准备工作
* 根据方便访问注册coze.cn或coze.com
* 生成token
* 下载xExtension插件
* 通过Tools->Create Agent Wizard选择coze网站，token和space
* 选择要创建的bot（Agent），点击Create Bot，等待创建结束
* 后期可以通过Tools->Config Agent来修改上述配置

## 模型创建
在创建模型文件的时候，如果已经安装配置好该模型的Agent，除文件名外会显示Description输入框，填入文字需求即可
